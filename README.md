# veewee-definitions-netbsd

[Veewee][veewee] definitions for NetBSD Vagrant base boxes.

If you use NetBSD under Vagrant you may also need the Vagrant
plugin [vagrant-guest-netbsd][vagrant-guest-netbsd].
